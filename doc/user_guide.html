User Guide

Main Window Layout

The main window comprises three major areas:
Menu Bar
Provides all the menu commands as documented in the section below

Puzzle Grid
This is where puzzles are displayed and solved

Status Bar
The status bar appears beneath the Grid and is divided into 5 fields from left to right:
  • A multi-purpose field showing either a:
    • Brief menu item help for the menu item under the mouse
    • Process message for tasks that could take a long time like creating, validating or grading puzzles
    • Histogram of the number of remaining values to solve
  • The state of play:
    • Idle
    • Enter: Manual entry of puzzles
    • Load: Loading a puzzles from file
    • Generate: Creating a puzzle
    • Validate: Validating and grading an entered or loaded puzzle
    • Solve: Solving a puzzle
  • The selected level of assistance (see Puzzle Menu below)
  • The graded difficulty of the puzzle at play
  • The solve duration timer

Menu Bar Commands

Puzzle Menu

Generate
Creates a puzzle up to the expertise level and with the symmetry selected under the Options menubar item

Enter
Enables/Disable (toggle) user entry of a puzzle

Load
Loads a puzzle from a file

Scramble
Scrambles an existing puzzle maintaining mathematical equivalence

Save
Saves a complete, partially entered or solved puzzle to disk

Show Possible Candidates
While solving show all possible candidates in empty cells

Clear Candidates
While solving, clear all candidate values

Pause
Pause solving toggle - hide the grid and pause the timer

Hint
While Solving:
  • Vague: Logic technique to solve next value
  • Clear: Logic technique details

List Solution
TBI: List the logic technique steps to solve the rest of the puzzle

Start Over
Start solving the same puzzle over again

Start New
Discard the current puzzle and clears the grid

Exit
Exit the Sudoku puzzle app

View Menu

Zoom
  • Tiny: Rediculously small grid
  • Small: Small grid
  • Normal: Normal grid
  • Large: Large grid
  • XLarge: Rediculously large grid

Unselected Candidates visible
Toggle the visibility of unselected candidates

Options Menu

Generate Expertise level
The expertise level request to generate puzzle. Generated puzzle maybe to less than requested level. Choice of Beginner, Novice, Intermediate, Proficient, Accomplished, Expert, or Guru

Grid Symmetry
A choice of:
  • Random: No symmetry
  • TBI: Dihedral: 8 point symmetry
  • TBI: Square Quadrant 90Deg rotation: 4 point symmetry
  • TBI: Square Quadrant Mirrored: 4 point symmetry
  • TBI: Diagonal Quadrant 90Deg rotation: 4 point symmetry
  • TBI: Diagonal Quadrant Mirrored: 4 point symmetry
  • TBI: Vertical Fold 180Deg rotation: 2 point symmetry
  • TBI: Vertical Fold mirrord: 2 point symmetry/li>
  • TBI: Horizontal Fold 180Deg rotation: 2 point symmetry
  • TBI: Horizontal Fold mirrored: 2 point symmetry
  • TBI: Diagonal top left fold 180Deg Rotation: 2 point symmetry
  • TBI: Diagonal top left fold mirrored: 2 point symmetry
  • TBI: Diagonal bottom left fold 180Deg Rotation: 2 point symmetry
  • TBI: Diagonal bottom left fold mirrored: 2 point symmetry

Value Assistance
A choice of:
  • Check On Finish: Indicate if puzzles solution is correct or not
  • Show Conflicts On Finish: Not necessarily the cell error
  • Show Errors On Finish
  • Show Conflicts: Interactively show conflicts
  • Show Errors: Interactively show errors

Candidate Assistance
Toggle. When enabled, hides shown peer candidate values of values entered

Keyboard and Mouse Input

Entry State

Mouse Left Button
Selects only the cell under cursor

Shift + Mouse Left Button
Adds cell under cursor to selected cells

[1-9]
Enters typed digit in selected cells as given digits

Shift + [1-9]
Enters typed digit in selected cells as placed digits

Del or BS(Ctrl + H)
Clears selected cells

Ctrl + Z
Undoes last step

Solve State

Mouse Left Button
Selects only the cell under cursor

Shift + Mouse Left Button
Adds cell under cursor to selected cells

[1-9]
Enters typed digit in selected cells that are not givens

Del or BS(Ctrl + H)
Clears selected cells that are not givens

Ctrl + Z
Undoes last step

Alt + Mouse Left Button
Background colour picker for selected cells

Ctrl + Mouse Left Button
Transient highlight of same values as under cursor

Ctrl + Alt + Mouse Left Button
Transient highlight of group under cursor

Solve State - Pencil Marks

Mouse Right Button
Toggle the visibility of the candidate value under cursor

Shift + Mouse Right Button
Selects the candidate under the cursor as the cell's value

Ctrl + Mouse Right Button
Transient highlight of all cells containing candidate value under cursor

Alt + Mouse Right Button
Colour picker for candidate under cursor

Ctrl + Alt + Mouse Right Button
Clear all candidate background colour selections

Cell Grammar

Location Specifiers

r6c5
The cell in row six, column five

b21
The cells in box/block two across and one down

r2
The cells in row two

c6
The cells in column six

r34c245
A grid of cells, r3c2, r3c4, r3c5, r4c2, r4c4 and r4c5

r2r5,r3c6,r56c56
A collection of cells in a comma separated list

Operators

--
Presence of candidate or value in cell(s)

!-
Absence of candidate(s) in cell(c)

?=
Possibility of the presence of candidate(s) (phantom) in cells(s)

==
Presence of only specified candidate(s) or value in cell(s)

!=
Value or candidate not permitted in cell(s)

:=
Assignment of a value or candidates to cell(s)

+=
Add candidate(s) to cell(s)

-=
Eliminate candidate(s) from cell(s)

?
Query or test a condition

#
Number of occurrences or count of candidate(s) or value in cell(s)

=
Strong link symbol

-
Weak link symbol

Examples

r6-2
Refers to value or candidate 2 in row 6

r4,b23-7
Refers to value or candidate 7 in row 4 and box 2 down, 3 across.

r6c2,r23c24-34
Refers to candidates 3 and 4 (amongst others) in the the collection of cells.

r5c3,r3c2?-5
Refers to the possiblity that 5 may be or may have been a (phantom) candidate in r5c3 and/or r3c2.

#2c3--5
There are two occurances in column three of candidate or value five.

c2=4=r2c2-5-r2c6
Links and chains
  • =<n>=: refers to n forming a strong link (slink) between two cells
  • -<n>-: refers to n forming a weak link (wink) between two cells.

r6c2,r23c24--34
The presence of both candidates 3 and 4 in the specified collection of cells

r6c2,r23c24!-34
Absence of both candidates 3 and 4 in the specified collection of cells

r6c2,r2c2==3
Three is the only value in that collection of cells

r6c2,r23c24!=3
Three is not (or cannot be) the value of that collection of cells

r6c2,r23c24:=34
Assign candidate values 3 and 4 to the collection of cells (replaces existing)

r6c2,r23c24!=34
Collection of cells do not contain candidates 3 and 4 exactly. This will be false if any cell in the collection contains additional values or is missing a 3 or 4.

r6c2,r23c24-=34
Eliminate candidate values 3 and 4 from the collection of cells

r6c2,r23c24+=34
Add candidate values 3 and 4 to the collection of cells
